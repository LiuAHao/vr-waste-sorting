# ParkClean VR 垃圾分类小游戏

ParkClean VR 是一个基于 Unity 的 VR 垃圾分类教育小游戏。项目当前由“清扫社区”Demo 重构而来，目标是把原本简单的拾取、清理、计数玩法升级为“观察垃圾、抓取垃圾、判断类别、投入垃圾桶、获得反馈、查看复盘”的完整垃圾分类体验。

项目面向校园、社区和公共环保教育场景，希望通过沉浸式 VR 交互降低垃圾分类学习门槛，让用户在接近真实生活的场景中反复练习常见垃圾分类规则，并将虚拟体验迁移到现实环保行为中。

## 核心体验

- 进入校园或社区生活场景。
- 通过 VR 手柄、射线或后续手势交互抓取垃圾。
- 根据垃圾外观、材质、污染程度和生活语境判断分类。
- 将垃圾投入可回收物、有害垃圾、厨余垃圾、其他垃圾四类垃圾桶。
- 系统即时判断正误，给出动画、音效、震动和文字解释。
- 任务结束后展示用时、正确率、错误物品和环保影响反馈。

## MVP 范围

首个可演示版本聚焦一个稳定闭环：

- 1 个校园食堂或社区投放点场景。
- 1 组四分类垃圾桶。
- 约 12 个高频垃圾物品，每类 3 个。
- 抓取、移动、投放、判定、反馈、重试。
- 倒计时、目标进度、结算页和错误复盘。
- 本地记录正确率、用时、错误类型等基础数据。

## 当前工程基础

- 引擎：Unity。
- 语言：C#。
- 当前玩法基础：已有清扫计数、倒计时、胜负面板、场景切换等早期脚本。
- 资产基础：城市场景资源、低多边形垃圾素材、四分类垃圾桶 GLB 模型、声音素材。
- 设计基础：`docs/design` 下已有总体设计、MVP 设计、模块拆分和建模清单。

## 推荐开发方向

1. 将现有“点击清扫垃圾”逻辑重构为“垃圾物品 + 分类属性 + 垃圾桶判定”逻辑。
2. 用数据配置管理垃圾名称、类别、难度、解释文案和模型引用。
3. 先完成非联网本地 Demo，再逐步补充 VR 交互、动画反馈、数据分析和多场景扩展。
4. 保持低眩晕、高帧率和轻量化资源策略，优先保障 VR 体验稳定。
