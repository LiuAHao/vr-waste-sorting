## 模块说明

本模块负责运行时 UI 的创建与展示。

当前包含：
- `WasteStartView`：开始游戏界面，展示目标、规则说明和开始按钮。
- `WasteHudView`：显示时间、得分、进度和即时投放反馈。
- `WasteResultView`：显示结算统计、错误记录和重新开始按钮。
- `WasteUiFactory`：统一创建 Canvas、面板、文本和按钮。

当前逻辑：
- 进入场景先显示开始页。
- 点击开始后才正式开始游戏计时。
- 游戏过程中显示 HUD。
- 正确投放后垃圾直接完成并消失。
- 错误投放后垃圾同样直接消失，并计入进度。
- 达成目标或超时后显示结算页。
