## 模块说明

这个目录存放的是 Unity 编辑器辅助工具，不参与运行时打包。

当前包含：
- `WasteContentMenu`：创建垃圾内容目录配置资产。
- `WasteSceneBuilderWindow`：把内容目录中的垃圾和垃圾桶批量拉入主场景，或对已经手动摆好的对象补齐交互组件。

当前约束：
- 项目只保留一个主场景：`Assets/Scenes/2.unity`
- 内容接入、资源验收、场景调试都在 `2.unity` 中完成
- 不再提供“恢复默认目录配置”的重置入口，避免误覆盖当前已调好的布局

推荐使用方式：
1. 执行 `垃圾分类项目/内容配置/创建内容目录配置`。
2. 打开 `垃圾分类项目/场景工具/主场景构建器`。
3. 如果场景里还没有摆放对象，就先打开主场景 `2.unity`，然后批量构建。
4. 如果场景里已经手动摆好了垃圾和垃圾桶，就使用“同步当前主场景对象交互配置”，只补交互，不改位置。
5. 手动调整完位置、旋转、缩放后，点击“将当前场景布局回写到目录配置”保存。
6. 后续再次批量构建时，会严格按目录配置里保存的位姿和缩放还原，不会再按主视角重新排布。
